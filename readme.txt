文件目录说明:
一.android目录
1.common.utils中是一些常用的工具类:包括字符转码,日志打印,加密.日期格式转换,http请求,json处理等

2.ui.utils中是一些android中常用的工具类,弹窗的工具类,Toast的工具类等

3.widget中是一些常用组件:其中pulltorefresh是一些下拉刷新的组件,
  switchview中包括两个滑动按钮的组件:SwitchButton和FrameAnimationController为一组,而Switch为另一组
  
  
二.zxing目录是二维码扫描,以及二维码和条形码生成的工具类


三.afinal是第三方的框架的源代码
